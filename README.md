# 멍백홈 · 유실목격지도

> 제보는 한곳에, 수색은 더 똑똑하게

두 가지 버전이 있습니다.

| 버전 | 파일 | 용도 |
| --- | --- | --- |
| **데모 (파일 하나)** | [유실목격지도.html](유실목격지도.html) | 더블클릭으로 열리는 시연용. 서버 불필요, 새로고침하면 리셋 |
| **실서비스형 (로컬 풀스택)** | [mungbaek/](mungbaek/) | Python 백엔드 + REST API + JSON DB. 제보·상태가 실제로 저장됨 |

## 실서비스형 실행 방법

```
cd mungbaek
py server.py
```

| 주소 | 화면 |
| --- | --- |
| http://localhost:8000/ | 보호자 · 유실목격지도 (핵심 화면) |
| http://localhost:8000/new | 보호자 · 유실 공고 등록 (강아지 정보 + 지도로 유실 위치 찍기) |
| http://localhost:8000/share | 보호자 · 공고 공유 (실제 QR 포함) |
| http://localhost:8000/r/kong-i | 목격자 · 제보하기 (앱 설치 없이) |

- 추가 설치 없이 Python 표준 라이브러리로 동작합니다. QR 생성만 `py -m pip install qrcode` 필요(이미 설치됨).
- 데이터는 `mungbaek/data.json`에 저장됩니다. **초기화하려면 이 파일을 지우고 서버를 재시작**하세요.

## 구현된 기능

- REST API: `GET/POST /api/notice`, `GET/POST /api/reports`, `PATCH /api/reports/:id`, `GET /api/qr`
- **유실 공고 등록**(`/new`): 강아지 정보 입력 + 지도로 유실 위치 찍기 → QR 자동 생성, 제보 초기화
- **목격자 제보 → 실시간 반영**: 서버 저장 후 보호자 지도에 2초 폴링으로 즉시 표시 (NEW 배지 + 토스트)
- **방향 표시(핀 vs 화살표)**:
  - 목격자: 나침반(폰이 바라보는 방향, `DeviceOrientation`) / 지도에서 방향 찍기 / 방향 없이 핀만
  - 방향이 있으면 지도에 **화살표 마커**, 없으면 **핀**으로 표시
- **보호자 직접 제보 추가**: 전화·SNS 등 외부로 받은 목격 정보를 지도를 탭해 직접 등록 (방향 8방위 + 신뢰 상태 선택)
- 브라우저 푸시 알림: 보호자 지도 "🔔 알림" 버튼 → 새 제보 도착 시 OS 알림
- 사진 제보: 업로드 시 900px 리사이즈 저장, 목록·상세에 표시
- 실제 스캔되는 QR 코드
- 제보 상태 관리(신뢰/확인 필요/숨김/중요) — 서버 영속 저장
- 이동 경로·우선 수색 구역: 실제 목격 시각 기준, 신뢰·중요 제보만 반영
- **카카오맵 실지도 연동 완료**: `config.js`에 JavaScript 키 입력 시 실제 카카오맵(핀·화살표·경로·수색구역 오버레이). 키 없거나 미승인 도메인이면 자동으로 SVG 도식 지도로 폴백

> 참고: 나침반 방향은 **모바일 + https**에서 동작합니다(데스크톱은 나침반 버튼이 숨겨지고 지도 방향 찍기로 대체). 배포(https) 후 폰에서 테스트하세요.

## 휴대폰으로 QR 시연하기

1. PC와 휴대폰을 같은 와이파이에 연결
2. PC의 IP 확인: `ipconfig` → IPv4 주소 (예: 192.168.0.10)
3. PC 브라우저에서 `http://192.168.0.10:8000/share` 를 열면 QR이 그 주소 기준으로 생성됨
4. 휴대폰 카메라로 QR 스캔 → 제보 페이지가 열림 → 제보하면 PC의 보호자 지도에 실시간으로 뜸

## 시연 시나리오 (약 2분)

1. `/share` — 공고 포스터와 QR을 보여주며 "보호자가 공고를 만들어 뿌립니다"
2. `/r/kong-i` — "목격자는 앱 설치 없이 엽니다". 빠른 선택 칩 + 시각·메모 입력 → 제보 보내기
3. `/` — 4초 안에 주황색(확인 필요) 새 핀 + NEW 배지 + 토스트 (알림을 켜뒀다면 OS 알림도)
4. 새 제보 클릭 → **신뢰** → 이동 경로 연장 + 우선 수색 구역 갱신
5. 17:50 놀이터 제보(숨김)로 오인 제보 걸러내기 설명
